Luxury Dining Menu Display

A responsive React application showcasing a high-end restaurant menu. Built with Vite, it features a "Luxury Dark Theme" aesthetic, a responsive grid layout for menu items, and reusable components.

This project uses a modular component architecture:

App.jsx: The main container that holds the page layout and menu data.

Header.jsx: Displays the restaurant branding, title, and spotlight effects.

MenuGrid.jsx: A container component that manages the CSS Grid layout to ensure responsiveness across devices.

MenuItem.jsx: The core reusable component. It accepts props (image, name, price, description, isVegetarian, isChefsChoice) and renders the distinct "Box" card with hover animations.

Footer.jsx: Displays copyright and footer links.
